About the project

Docs Drive is really a clone of Google Docs. It's a text editor which saves your work automatically as you write. App keeps your files in built-in file explorer that you have whole access to.

Since the project has some essential global data, I deciced to go with redux to share crucial information with every component so I don't need to deal with passing props which makes no sense later with extending the app
